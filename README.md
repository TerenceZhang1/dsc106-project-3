# Cloud Cover Over California Counties

An interactive data visualization project exploring cloud coverage across California counties using GOES-18 satellite cloud mask data.

This project was created for DSC 106 as a cloud cover data visualization project.

## Overview

This project investigates the question:

**How does cloud coverage vary across California counties over multiple daytime hours?**

Through interactive county-level maps, the project shows how cloud cover changes across California over several days. Users can explore cloud coverage over time with a timeline view or compare two different timestamps side by side.

The visualization includes county-level cloud cover percentages, hover tooltips, synchronized county highlighting, county zooming, major city labels, and date markers to make the data easier to interpret.

## Data Used

This project uses **GOES-18 ABI-L2-ACM cloud mask data** from November 3-5, 2025.
